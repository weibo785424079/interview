- 其它
  - module、chunk、bundle 的区别
    - module 各个源码文件，webpack 中一切皆模块
    - chunk 多模块合成的一个代码产出块，如 entry、import()、splitChunk 等
    - bundle 最终产出的文件，每个 chunk 打包完都会产生一个 bundle
